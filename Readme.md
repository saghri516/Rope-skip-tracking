# Smart Rope Fitness Tracker PRO

Real-time AI-based jump rope tracking system using computer vision.

## Features
- Jump detection using pose estimation
- Real-time timer (session based)
- RPM (jumps per minute)
- Calories estimation
- Workout history logging

## Tech Stack
- Python
- OpenCV
- MediaPipe
- CSV logging

## Run
```bash
pip install -r requirements.txt
python main.py
